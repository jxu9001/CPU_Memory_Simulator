Included Files:
CPU.java (simulates the CPU)
Memory.java (simulates the memory)
sample5.txt (displays the first 15 fibonacci numbers)

How to compile and run the files:
1. navigate to the directory containing the .java files and the .txt file(s)
2. type the following into the terminal (no quotes)
3. "javac *.java"
4. "java CPU <inputFile> <timerValue>"
